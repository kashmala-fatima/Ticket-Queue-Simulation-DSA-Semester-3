# 🎫 Online Ticket Management System (DSA)

A C++ application using **Queue Data Structure** and **OOP** to simulate an online ticketing process.

## ✨ Key Features
* **FIFO Logic:** Customers are served in the order they arrive.
* **Ticket Booking:** Simple interface to add and process ticket requests.
* **OOP Design:** Modular code with classes for easy maintenance.

## 🛠️ Built With
* C++
* Code::Blocks IDE
